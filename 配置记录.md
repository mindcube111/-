## 📝 配置步骤

### 1. 在 Cloudflare Dashboard 配置环境变量

**访问路径：**
- Workers & Pages → Pages → stranger-things-test → Settings → Environment variables

**需要添加的变量：**

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `ADMIN_PASSWORD` | `你的管理员密码` | 管理员登录密码 |
| `INVITE_SALT` | `上面选择的盐值` | 邀请码哈希盐值 |

### 2. 绑定 KV 命名空间

**访问路径：**
- Workers & Pages → Pages → stranger-things-test → Settings → KV namespaces

**绑定配置：**
- Variable name: `INVITE_CODES`
- KV namespace: 选择已创建的 `INVITE_CODES` 命名空间

## ⚠️ 重要提示

1. **盐值选择后不要更改**
   - 如果更改 `INVITE_SALT`，之前生成的邀请码将无法使用
   - 需要重新生成所有邀请码

2. **保存配置记录**
   - 记录下选择的盐值
   - 记录下管理员密码（安全保存）
   - 记录下 KV Namespace ID

3. **配置完成后测试**
   - 测试管理员登录
   - 测试生成邀请码
   - 测试验证邀请码
