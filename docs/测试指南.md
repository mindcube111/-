# 怪奇物语性格测试 - 测试指南

## 🚀 快速开始

### 1. 打开项目

直接用浏览器打开 `index.html` 文件即可开始使用。

### 2. 管理员首次登录

1. 打开 `admin.html`
2. 输入默认密码：`<ADMIN_PASSWORD>`
3. 生成邀请码并分发给测试用户

### 3. 用户测试流程

1. 打开 `index.html`
2. 输入管理员提供的6位邀请码
3. 完成60道选择题
4. 查看专属性格分析报告
5. 可选择保存结果或重新测试

## 🔧 技术说明

### 前端技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6)** - 交互逻辑
- **Chart.js** - 雷达图绘制
- **html2canvas** - 结果保存

### 数据存储

所有数据存储在浏览器的 `localStorage` 中：

- `device_id` - 设备唯一标识
- `invite_codes` - 邀请码列表
- `test_answers` - 测试答案
- `test_result` - 测试结果
- `admin_password` - 管理员密码

### 设备识别

设备ID = `navigator.userAgent + 语言 + 屏幕分辨率 + 时区 + UUID`

**注意**：用户清除浏览器缓存可绕过设备限制。

## 🎯 隐藏款触发条件

要触发隐藏角色"达达"，用户需要：

1. 第27题选择 **A** （收养特别的小生物）
2. 第40题选择 **A** （坚持留下误伤人的宠物）

同时满足两个条件，且计算结果中"达达"匹配度最高时，才会显示隐藏款。

## ⚠️ 注意事项

### 管理员

1. 妥善保管管理员密码
2. 定期检查邀请码使用情况
3. 及时删除已用完的邀请码

### 用户

1. 请在良好的网络环境下完成测试
2. 不要刷新页面，答题进度会自动保存
3. 清除浏览器缓存会丢失所有数据
4. 每个邀请码在一个设备上最多使用2次

### 数据安全

- 所有数据仅存储在用户本地浏览器
- 不涉及任何服务器传输
- 管理员密码明文存储在localStorage（仅用于简单访问控制）

## 🐛 已知限制

1. **设备识别不够严格**：用户清除缓存可绕过限制
2. **无后端支持**：数据无法跨设备同步
3. **管理员密码简单**：仅前端验证，无加密
4. **邀请码无过期机制**：生成后永久有效

如遇到问题：

1. 检查浏览器控制台的错误信息
2. 确认浏览器支持localStorage
3. 尝试在隐私/无痕模式下测试
4. 清除浏览器缓存后重试
