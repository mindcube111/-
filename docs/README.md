# 《怪奇物语》性格测试

基于《怪奇物语》剧中人物性格设计的交互式性格测试，通过60道情景选择题评估你的行为模式、思维方式、情感反应和价值取向。

## 🎯 功能特点

- **60道精心设计的测试题**：涵盖日常场景与剧情核心场景
- **20个经典角色**：包含霍珀、南希、史蒂夫、十一、威尔等核心角色
- **隐藏款角色**：达达（需要特定答题条件触发）
- **科学计分系统**：基于多维度性格分析
- **八维雷达图**：可视化展示性格特质
- **精美UI设计**：怪奇物语主题的红黑配色

## 🚀 快速开始

### 方法一：直接打开
1. 双击打开 `index.html` 文件
2. 使用现代浏览器（Chrome、Safari、Edge等）打开

### 方法二：本地服务器
```bash
# 进入项目目录
cd "怪奇物语测试"

# 使用Python启动简单HTTP服务器（Python 3）
python3 -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000

# 然后在浏览器中访问
# http://localhost:8000
```

## 📊 评估维度

测试评估以下八个性格维度：

1. **行动力** - 面对危机时的决策速度与执行力
2. **合作性** - 团队协作与沟通能力
3. **守护欲** - 对重要之人的保护意识
4. **成就欲** - 追求目标与自我实现的动力
5. **稳定性** - 情绪调节与压力应对能力
6. **共情力** - 理解他人情绪与感受的能力
7. **逻辑性** - 理性分析与策略规划能力
8. **直觉性** - 凭借第六感做出快速判断的能力

## 💡 使用建议

1. **独立完成**：在安静环境下独立完成，避免他人干扰
2. **第一直觉**：选择最符合你真实反应的选项，而非你认为"应该"选择的选项
3. **完成时间**：预计用时 15-20 分钟
4. **结果参考**：测试结果仅供娱乐参考，不代表心理诊断

## 🎨 技术实现

- **前端技术**：纯HTML + CSS + JavaScript
