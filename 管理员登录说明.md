# 🔐 管理员登录说明

## 📋 登录信息

### 账号和密码配置

**管理员账号和密码**都需要在 Cloudflare Pages 的环境变量中配置：

1. 登录 Cloudflare Dashboard
2. 进入 Workers & Pages → Pages → stranger-things-test
3. 进入 Settings → Environment variables
4. 添加环境变量：
   - **变量名**: `ADMIN_USERNAME`
   - **值**: 你的管理员账号（例如：`admin`）
   - **变量名**: `ADMIN_PASSWORD`
   - **值**: 你的管理员密码（例如：`admin123`）

### 登录方式

- **账号**: 必须填写，与 Cloudflare 环境变量 `ADMIN_USERNAME` 一致
- **密码**: 必须填写，与 Cloudflare 环境变量 `ADMIN_PASSWORD` 一致

**注意**: 账号和密码都必须正确才能登录成功。

## 🎨 登录表单布局

登录表单已设置为**竖排显示**：
- 账号输入框（顶部）
- 密码输入框（中间）
- 登录按钮（下方）
- 退出登录按钮（最下方，登录后显示）

所有元素垂直排列，宽度一致。

## ⚙️ 配置步骤

### 1. 设置环境变量

在 Cloudflare Dashboard 中：

```
Settings → Environment variables → Add variable
```

添加两个环境变量：

**第一个变量：**
- **Variable name**: `ADMIN_USERNAME`
- **Value**: 你的管理员账号（例如：`admin`）

**第二个变量：**
- **Variable name**: `ADMIN_PASSWORD`
- **Value**: 你的管理员密码（例如：`MySecurePassword123`）

### 2. 保存并重新部署

环境变量修改后，需要重新部署才能生效：

```bash
wrangler pages deploy . --project-name=stranger-things-test
```

或者通过 Git 推送触发自动部署。

## 🔒 安全建议

1. **使用强密码**：至少8位，包含字母、数字和特殊字符
2. **定期更换密码**：建议每3个月更换一次
3. **不要分享密码**：管理员密码应保密
4. **使用环境变量**：不要将密码硬编码在代码中

## ❓ 常见问题

### Q: 忘记账号或密码怎么办？
A: 在 Cloudflare Dashboard 中修改 `ADMIN_USERNAME` 或 `ADMIN_PASSWORD` 环境变量，然后重新部署。

### Q: 账号和密码是什么？
A: 账号和密码都需要在 Cloudflare Pages 的环境变量中配置：
   - `ADMIN_USERNAME`: 管理员账号
   - `ADMIN_PASSWORD`: 管理员密码
   两者都必须正确才能登录。

### Q: 登录后看不到功能？
A: 检查浏览器控制台是否有错误，确保环境变量已正确配置。

### Q: 如何修改账号或密码？
A: 在 Cloudflare Dashboard 中修改 `ADMIN_USERNAME` 或 `ADMIN_PASSWORD` 环境变量值，保存后重新部署。

### Q: 默认账号密码是什么？
A: 没有默认值，必须在 Cloudflare Pages 环境变量中配置。首次部署时请务必配置这两个环境变量。

---

**重要提示**: 首次部署时，请务必在 Cloudflare Pages 中配置以下两个环境变量，否则无法登录：
- `ADMIN_USERNAME`: 管理员账号
- `ADMIN_PASSWORD`: 管理员密码
